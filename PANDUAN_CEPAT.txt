📌 CARA MENJALANKAN
─────────────────────────────────────────────────────────────────
  1. Double-click "JALANKAN_SEBAGAI_ADMIN.bat"
  2. Klik "Yes" pada UAC prompt
  3. GUI akan muncul dengan tombol-tombol menu!

📌 TOMBOL MENU GUI (Klik tombol sesuai kebutuhan)
─────────────────────────────────────────────────────────────────
  🔒 Aktifkan Kunci Instalasi (HIJAU)
      → User biasa TIDAK bisa install software
      → Harus pakai password Admin untuk install
  
  🔓 Nonaktifkan Kunci Instalasi (MERAH)
      → Kembalikan ke normal
  
  👤 Buat Akun Administrator Baru (BIRU)
      → Buat admin baru dengan nama & password pilihan Anda
      → Muncul form isian
  
  ⚙️ Aktifkan Administrator Built-in (KUNING)
      → Nyalakan akun "Administrator" bawaan Windows
      → Set password baru
  
  🔑 Ubah Password User (UNGU)
      → Ganti password user yang ada
      → Pilih user dari dropdown
  
  🔄 Refresh Status (PUTIH)
      → Lihat status terkini di panel kanan
      → Cek daftar admin & status kunci

📌 ALUR KERJA RECOMMENDED
─────────────────────────────────────────────────────────────────
  STEP 1: Double-click JALANKAN_SEBAGAI_ADMIN.bat
          GUI akan muncul
  
  STEP 2: Klik tombol BIRU (Buat Admin Baru)
          Isi form: nama "AdminUtama" + password
  
  STEP 3: Klik tombol HIJAU (Aktifkan Kunci)
          Instalasi sekarang terkunci!
  
  STEP 4: Klik tombol PUTIH (Refresh Status)
          Lihat konfirmasi di panel kanan
  
  STEP 5: Test dengan user biasa
          Coba install software → harus minta password Admin!

📌 KEAMANAN PASSWORD
─────────────────────────────────────────────────────────────────
  ✓ Minimal 8 karakter
  ✓ Kombinasi huruf besar + kecil + angka + simbol
  ✓ Jangan gunakan: 12345, password, admin, nama sendiri
  ✓ Contoh bagus: Admin@2024!, SecureP@ss99
  ✓ SIMPAN di tempat aman - JANGAN LUPA!

📌 TROUBLESHOOTING
─────────────────────────────────────────────────────────────────
  ❌ "Harus dijalankan sebagai Administrator"
     → Klik kanan file .bat → Run as Administrator
  
  ❌ Lupa password Administrator
     → Boot Safe Mode → Login admin lain → Reset password
  
  ❌ Tidak bisa aktifkan Administrator built-in
     → Pakai menu [3] buat admin baru saja

📌 TIPS PRO
─────────────────────────────────────────────────────────────────
  💡 Buat MINIMAL 2 akun admin (backup!)
  💡 Test dulu sebelum deploy ke banyak komputer
  💡 Catat password di password manager
  💡 Untuk warnet/lab: ganti password rutin (1 bulan sekali)

📌 CARA MEMBATALKAN SEMUA
─────────────────────────────────────────────────────────────────
  1. Jalankan tool (double-click .bat)
  2. Klik tombol MERAH - Nonaktifkan Kunci Instalasi
  3. Done! Kembali seperti semula
